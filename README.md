## üì¶ Overview

The **gcp-gitops-lighthouse-config** project provides a simple way to manage your cloud infrastructure using GitOps principles. It helps you set up a professional-grade environment on Google Cloud Platform (GCP). This project showcases how to use declarative infrastructure and automated continuous delivery.

## üí° Key Features

- **K3s Cluster:** Run a lightweight Kubernetes cluster for your applications.
- **ArgoCD:** Automate your continuous delivery process.
- **Terraform Modules:** Easily create and manage cloud resources.
- **Reproducible Environments:** Ensure consistency across different setups.

## üéì Usage Instructions

Once you have the application installed, you will need to configure it for your needs.

1. **Set Up Your Cloud Account:**
   Make sure you have a Google Cloud account. You will need this to create resources.

2. **Configure the Environment:**
   - Open the application.
   - Specify the settings for your GCP account.
   - Select the resources you want to manage using Terraform.

3. **Deploy Resources:**
   Use the interface to deploy your resources. The application provides clear options to guide you through this.

4. **Monitor Your Cluster:**
   Check the status of your K3s cluster through the application interface. This allows you to manage and scale your services as needed.
